# Section Title Arrow Collapse

This update removes the separate Expand/Collapse button from every section control bar.

Updated files:
- src/pages/Home.tsx
- src/index.css

Behavior:
- Each section now has an arrow on the left side of the section title.
- Clicking the section title or arrow expands/collapses that section.
- Export PNG and Top Nav buttons remain on the right side.
- The global Expand All / Collapse All button remains in the top dashboard navigation.
- All previous collapsed-by-default behavior is preserved.
