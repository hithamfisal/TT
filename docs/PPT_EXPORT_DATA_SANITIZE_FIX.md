# PPT Export Data-Safe Repair Fix

This package restores the original PPT formatting/charts from the theme-toggle version and applies a safer repair fix for PowerPoint corruption.

## Why this fix is different

The previous ultra-safe fix removed native charts/tables, which changed the report formatting. This version keeps the existing PowerPoint design, native charts, and native tables.

## Changes

Updated `src/pages/Home.tsx` only:

- Added `safePptText()` to remove illegal XML/control characters from workbook data before inserting into PPTX.
- Added `safePptTableRows()` so native PPT tables keep the same styling but receive safe text values.
- Kept the original PPT charts and dashboard report formatting.
- Replaced percentage slide-width values with numeric slide width for safer PPTX XML.
- Removed only the unsupported `rectRadius` custom property.

## Test

After extracting:

```powershell
yarn install
yarn run check
yarn build
```

Then export a new Monthly Tickets PPT and a new Performance PPT. Do not test previously exported files because those were generated by the older code.
