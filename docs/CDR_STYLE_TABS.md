# CDR-style Dashboard Tabs

This update arranges the existing dashboard sections as tabs without changing the internal layout of each section.

Updated files:
- src/pages/Home.tsx
- src/index.css

Behavior:
- Top section navigation now works as tabs.
- One section is visible at a time.
- The active tab is highlighted.
- Section export PNG and Top Nav remain available for the active section.
- Existing dashboard content/layout inside each section is preserved.
- Print mode shows all section content.
